Hello i am OpenXRAPI or Maxwell I make mods in my free time. This mod is completely free to use! it just requires Melon Loader!



  Menu

  - F1 opens and closes the menu.
  - Draggable menu header.
  - Protect/Misc switches between menu pages.
  - Saved settings automatically reload.

  Client Hacks
  
  - Block Other Player Movement (Blocks Photon event code 201/Photon status code 1042/DisconnectByServerUserLimit. )

  Movement

  - Super Slopfest Flight
  - Adjustable flight speed.
  - Customizable flight key.
  - Directional flight controls.
  - Handles gravity, collision, and Rigidbody state.
  - Restores original physics when disabled.
  - Zoomies
  - Adjustable Zoomies speed.
  - Directional fast movement while retaining gravity and colliders.
  - Refresh Player
  - Finds Player(Clone) only on the MyPlayer layer.
  - Renames it to YourPlayer(Clone).
  - Renames its PlayerMesh child to YourPlayerMesh.

  Teleportation

  - Click TP
  - C is the only toggle key.
  - Right-click teleports to the selected world position.
  - Stops unwanted physics velocity after teleporting.
  - Teleshitter
  - / toggles it.
  - Searches for objects by name.
  - Finds and tracks matching objects.
  - Teleports selected objects to your player.
  - Clear Object button.
  - Blocks player-like objects from being teleported.

  ESP Object Finder

  - Exact-name object searches.
  - Multiple saved object names.
  - Saved searches automatically reload.
  - Searches refresh every second.
  - Automatically highlights newly spawned matching objects.
  - Removes ESP when an object disappears.
  - Manual Refresh Now button.
  - Clear ESP button.
  - Cyan box ESP.
  - Object-name labels.
  - Pixel distance from the center of the screen.
  - Uses the ESP box center for targeting.
  - Adjustable ESP Y offset.
  - Dynamic Y offset based on world distance.
  - Stable character, collider, and renderer center detection.
  - Filters weapons, trails, effects, particles, shadows, decals, ground, and feet.
  - Rejects invalid or extreme screen projections.

  Aimbot

  - G toggles Aimbot.
  - Hold RMB to activate it.
  - Selects the ESP box nearest the center of the screen.
  - Targets the center of the ESP box.
  - Keeps the same target while RMB remains held.
  - Rotates the player body horizontally.
  - Rotates the gameplay camera toward the target.
  - Uses screen-center distance instead of player-to-object distance.
  - Works with multiple saved object types.
  - Aimbot enabled state is saved.
  - With Anti-ESP Spinbot enabled, Aimbot moves only the camera.

  Anti-ESP Spinbot

  - H toggles it.
  - Rapidly rotates YourPlayer(Clone).
  - Adjustable speed from 360 to 7200 degrees per second.
  - Keeps the Unity camera isolated from the rotation.
  - Allows independent camera movement and aiming.
  - Spin speed is saved.

  Local Client Protections

  - Default-on local protection watchdog.
  - Rejects NaN and Infinity transforms.
  - Recovers extreme local coordinates.
  - Recovers invalid player and camera rotations.
  - Stops invalid or extreme Rigidbody velocity.
  - Repairs invalid camera field of view.
  - Repairs invalid near and far clipping planes.
  - Filters invalid ESP coordinates.
  - Contains ESP overlay exceptions.
  - Session protection-event counter.
  - Refresh Safe State button.
  - Reset Event Count button.
  - Protection setting is saved.

  Emergency Reset

  - Press F8 or use the menu button.
  - Disables flight.
  - Disables Zoomies.
  - Disables Click TP.
  - Disables Spinbot.
  - Clears the active Aimbot target.
  - Restores flight physics.
  - Restores the last safe player position and rotation.
  - Restores the last safe camera position and rotation.
  - Stops local player velocity.
